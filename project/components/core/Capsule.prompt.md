Signature pill chip — use for event facts (date, time, location) and inline status labels. Smaller and lighter than a Button; not clickable.

```jsx
<Capsule tone="gold" iconLeft={<Clock size={14} />}>8:00AM Each Day</Capsule>
<Capsule tone="outline">Yota East, Legon</Capsule>
```

Tones: `purple`, `purple-soft`, `gold`, `magenta`, `outline` (for purple/photo backgrounds), `outline-ink`. Use `outline`/`outline-light` tones on dark fields.
