Standard section header — magenta/gold eyebrow above an uppercase Montserrat title. Opens nearly every page section.

```jsx
<SectionHeading eyebrow="What We Do" title="Our Four Pillars" />
<SectionHeading eyebrow="Theme" title="Into Reality" swoosh swooshWord="Turning Dreams" onDark />
```

Use `onDark` on purple sections (white title, gold eyebrow). `swoosh` draws the gold underline under `swooshWord`.
